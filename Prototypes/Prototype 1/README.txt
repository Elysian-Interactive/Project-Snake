The aim of this prototype is to:
1. Represent a snake with a linked list
2. Create a node to represent the snake body
3. The node must have the following feilds:
	a. The position : x, y
	b. The dimensions: w, h
	c. The collider
	d. Orientation: UP, DOWN, LEFT, RIGHT
4. Move around the snake with arrow keys
5. Press a button to grow the snake
